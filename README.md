# Python Thread-Safe Counter

Учебный проект на Python, показывающий, как безопасно работать с общими данными в многопоточном коде.

## Описание

В проекте реализован счётчик `Counter`, который увеличивается из нескольких потоков.  
Для защиты значения используется `threading.Lock`, чтобы избежать race condition.

## Что демонстрирует проект

- создание потоков через `threading.Thread`;
- передачу аргументов в поток;
- ожидание завершения потоков через `join()`;
- использование `threading.Lock`;
- безопасное изменение общего состояния.
